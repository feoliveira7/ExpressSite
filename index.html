O que é o Express?

O Express é um framework minimalista para Node.js, que facilita a criação de servidores e APIs. Com ele, é possível criar aplicações web de forma rápida e sem muita complexidade.

Vantagens do Express

  • Minimalismo: O Express é minimalista e permite que você adicione apenas o que realmente precisa para o seu projeto.
  • Roteamento: Facilita o roteamento de URLs e a manipulação de requisições HTTP.
  • Extensível: O Express pode ser facilmente estendido com middleware para adicionar funcionalidades como autenticação, logging, entre outros.
  • Comunitário: O Express tem uma grande comunidade, com muitos tutoriais e pacotes disponíveis para resolver diversos problemas.

Exemplo Básico de Código

            
const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Olá, Express!');
});

app.listen(3000, () => {
    console.log('Servidor rodando na porta 3000');
});

Este é um exemplo simples de como criar um servidor Express que responde com "Olá, Express!" quando a rota raiz ("/") é acessada.
